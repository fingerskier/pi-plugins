# Theology Pi Plugin

Pi package port of [`theology-claude-plugin`](https://github.com/fingerskier/theology-claude-plugin).

Provides Agent Skills for rigorous, multi-perspective Biblical exegesis.

## Install

```bash
pi install npm:@fingerskier/pi-theology
# local workspace clone
pi install ./packages/pi-theology
# one run only
pi -e npm:@fingerskier/pi-theology
```

## Skills

- `/skill:historian` — historical and cultural background
- `/skill:linguist` — Hebrew, Greek, Aramaic linguistic analysis
- `/skill:author` — literary analysis
- `/skill:theologian` — theological themes and doctrines
- `/skill:disciple` — hermeneutical/canonical analysis
- `/skill:shepherd` — practical application
- `/skill:research` — full exegetical analysis

No extension or external runtime is required.
